Introduction to css media queries

Media Query is a css technique that uses the @media rule to include a block of css properties only if a certain condition is true.

we write that code like this
@media screen and (here we put the condition for a break point for example, max-width:600px)

This technique helps programmers deliver a specific style sheet, to desktops, laptops, and mobile phones.

screen sizes

media queries can be used to check, width and height of the viewport, orientation of the viewport and also the resolution


Recommended to design mobile first

since mobile phones are now so common, and people use them more easily, its best when designing a web application, that the programmer starts with the design for such, and then later create a media query that can apply to lager screen sizes

Recommended Device Breakpoints

Extra small devices (phones, 600px and below)

@media only screen and (max-width: 600px) {...}

small devices (portrait tablets and large phones, 600px and above)

@media only screen and (min-width: 600px) {...}

medium devices (landscape tablets, 768px and above)

@media only screen and (min-width: 768px) {...}

large devices (laptops/desktops, 992 and up)

@media only screen and (min-width: 992px) {...}

Extra large devices (large laptops and desktops, 1200px and up)

@media only screen and (min-width: 1200px) {...}

For orientation: portrait/landscape

@media only screen and (orientation:landscape) {...}

Testing on different screen sizes

It is important to test on different screen sizes before deploying, as we know that a programmer needs to ensure that his/her application is user friendly for all users with different devices.
